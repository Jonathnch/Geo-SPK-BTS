# Geo-SPK: Sistem Pendukung Keputusan Pemerataan BTS

Aplikasi berbasis Web-GIS menggunakan algoritma **Fuzzy C-Means** dan **Streamlit** untuk memetakan kesenjangan infrastruktur telekomunikasi seluler (BTS) di Indonesia secara multidimensional.

Silakan klik tautan berikut untuk langsung menggunakan program tanpa perlu melakukan instalasi:  
**[Link Program](https://geo-spk-bts-u658nswoxjcmr26cjf2dft.streamlit.app/)**

---

### 🗂️ Pemenuhan Kelengkapan Berkas Skripsi
Repositori ini disusun untuk memenuhi kelengkapan penilaian skripsi:

1. **Program Siap Digunakan:** Tersedia dalam bentuk Web App melalui tautan di atas, dan script Python yang siap dieksekusi.
2. **Listing Program (Source Code):** - `app.py` (Script antarmuka Web-GIS)
   - `algoritma.py` (Script komputasi Fuzzy C-Means & Feature Engineering)
3. **File Instalasi Program:** Tersedia pada file `requirements.txt`.
4. **Manual Pemakaian Program:** - Panduan langsung: [Buka MANUAL.md](MANUAL.md)
